# Cliente de ejemplo — Perfil

Plantilla de **perfil de cliente**. Cada cliente tiene su propia subcarpeta dentro de `clientes/` con la información que necesitas a mano al trabajar con él.

## Datos

- **Nombre:** Cliente de Ejemplo S.L.
- **Contacto:** nombre de la persona · correo · teléfono
- **Sector:** el que sea

## Contexto

De qué va el cliente, qué espera, cómo prefiere que se le hable, plazos y cualquier cosa que convenga recordar de una sesión a otra.

## Trabajos

- Descripción breve de lo que se le hace o se le ha hecho.
