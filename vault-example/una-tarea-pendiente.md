# Migrar las notas antiguas al formato nuevo

Ejemplo de **pendiente**: una tarea sin fecha, "para cuando haya tiempo". Vive en la **raíz** de la bóveda, a la vista, marcada con `#pendiente` para que el router la vea al arrancar.

Cuando se cierre, esta nota se mueve a su sitio definitivo (o se borra si ya no aplica). No se queda en la raíz para siempre.

Si la tarea tuviera **fecha** concreta, no iría aquí: iría en `programado/` con `APARECER`. Ver [[recordatorio-ejemplo]].
